# Changelog

## v1.0.0 2017-10-03

  * `SMTPServerAsPromised` no more extends `SMTPServer`. The `SMTPServer`
    object is stored in `server` property.

  * Typings for Typescript.
  * Based on promise-readable@1.x.x
